About Me

I'm a Ph.D. student in the Department of Computer Science. My research interest lies in the fields of Robotics and Data Science. I had developed two robot motion planning algorithm on integrating machine learning techniques during master's program. I'm currently working on topological path planner which uses Topological Data Analysis (TDA) method to analyse the metric space properties for successful path planning in a dynamic environemnt.
